# 学生作业包

这个目录是课程对学生发放的统一作业包，包含作业说明、固定数据集、基线脚本和参考结果。

## 三个作业

1. `Q1`：围绕共享输入场景下的推理吞吐优化，目标是提高 `prompt_tokens_per_sec`。
2. `Q2`：围绕单请求延迟优化，目标是在固定 workload 上降低 `mean_latency_sec`。
3. `Q3`：围绕 Ascend 平台上的运行时与并行配置优化，目标是提高 `output_tokens_per_sec`。

## 使用顺序

1. 阅读 `problems/` 目录下对应题目的说明文件。
2. 保持 `data/` 目录下的数据集内容不变。
3. 从 `scripts/` 目录下的基线脚本开始实验。
4. 使用 `artifacts/` 中的基线结果作为对照。

## 目录说明

- `login/`：登录说明与辅助脚本
- `problems/`：题面、限制条件、评分指标
- `data/`：固定数据集
- `scripts/`：基线脚本，统一使用 `LLM(...)`
- `artifacts/`：基线结果参考
